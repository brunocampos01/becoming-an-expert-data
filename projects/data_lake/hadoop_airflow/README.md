> This is an example of data lake using Apache Hadoop technologies in on-premises environment

# Stack of Technologies
## Resource Management
- Cloudera Manager
- Apache Zookeeper
- Apache YARN

## Storage
- Oracle Database: source data
- HDFS: layer of data lake
- Apache Hive
- Apache Impala
- Redis: cache

## Data Processing
- Hadoop mapReduce
- Apache Spark
- Pandas
- Redis

## Scheduler
- Apache Airflow

## Data Types
- tables
- avro
- blobs

## Infra as Code
- Shell Scripts
- Ansible
- Docker

## Security
- Kerberos
- Apache Sentry
- ACL of SO and HDFS
